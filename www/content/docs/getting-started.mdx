## Prerequisites

- Node.js 18 or higher
- npm, pnpm, or yarn

## Create a project

<Steps>
  <Step>
    ### Run the CLI

    ```bash title="Terminal"
    npx pest-js-app
    ```

    You'll be asked to configure your project:

    1. **Project name** (kebab-case, e.g. `my-api`)
    2. **Description**
    3. **Author** (pulled from your git config)
    4. **Database ORM** (Prisma, Drizzle, TypeORM, or None)
    5. **Database provider** (PostgreSQL, MySQL, or SQLite)
    6. **Docker support**
    7. **Package manager** (npm, pnpm, or yarn)
    8. **Git init**
    9. **Install dependencies**
  </Step>
  <Step>
    ### Start the dev server

    <Tabs defaultValue="npm">
      <TabsList>
        <TabsTrigger value="npm"><NPM className="size-4" /> npm</TabsTrigger>
        <TabsTrigger value="pnpm"><Pnpm className="size-4" /> pnpm</TabsTrigger>
        <TabsTrigger value="yarn"><Yarn className="size-4" /> yarn</TabsTrigger>
      </TabsList>
      <TabsContent value="npm">
        ```bash title="Terminal"
        cd my-api
        npm run dev
        ```
      </TabsContent>
      <TabsContent value="pnpm">
        ```bash title="Terminal"
        cd my-api
        pnpm dev
        ```
      </TabsContent>
      <TabsContent value="yarn">
        ```bash title="Terminal"
        cd my-api
        yarn dev
        ```
      </TabsContent>
    </Tabs>

    Your API is now running at `http://localhost:3000` with hot reload via `tsx watch`.
  </Step>
  <Step>
    ### Build for production

    <Tabs defaultValue="npm">
      <TabsList>
        <TabsTrigger value="npm"><NPM className="size-4" /> npm</TabsTrigger>
        <TabsTrigger value="pnpm"><Pnpm className="size-4" /> pnpm</TabsTrigger>
        <TabsTrigger value="yarn"><Yarn className="size-4" /> yarn</TabsTrigger>
      </TabsList>
      <TabsContent value="npm">
        ```bash title="Terminal"
        npm run build
        npm start
        ```
      </TabsContent>
      <TabsContent value="pnpm">
        ```bash title="Terminal"
        pnpm build
        pnpm start
        ```
      </TabsContent>
      <TabsContent value="yarn">
        ```bash title="Terminal"
        yarn build
        yarn start
        ```
      </TabsContent>
    </Tabs>
  </Step>
</Steps>

## Default routes

Every project ships with these endpoints:

| Route | Method | Description |
|-------|--------|-------------|
| `/` | GET | Welcome message with API version and environment |
| `/health` | GET | Health check (status, timestamp, uptime) |
| `/api/examples` | GET | List all examples |
| `/api/examples` | POST | Create an example (validated with Zod) |
| `/api/examples/:id` | PUT | Update an example |
| `/api/examples/:id` | DELETE | Delete an example |
