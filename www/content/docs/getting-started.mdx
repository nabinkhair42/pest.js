## Prerequisites

- Node.js 18 or higher
- npm, pnpm, or yarn

## Create a project

<Steps>
  <Step>
    ### Scaffold

    Run the CLI with npx:

    ```bash title="Terminal"
    npx pest-js-app
    ```

    You'll be prompted for:

    1. **Project name** — kebab-case (e.g. `my-api`)
    2. **Description**
    3. **Author** — auto-detected from git config
    4. **Database ORM** — Prisma, Drizzle, TypeORM, or None
    5. **Database provider** — PostgreSQL, MySQL, or SQLite (if ORM selected)
    6. **Docker support**
    7. **Package manager** — npm, pnpm, or yarn
    8. **Git init**
    9. **Install dependencies**
  </Step>
  <Step>
    ### Start developing

    <Tabs defaultValue="npm">
      <TabsList>
        <TabsTrigger value="npm"><NPM className="size-4" /> npm</TabsTrigger>
        <TabsTrigger value="pnpm"><Pnpm className="size-4" /> pnpm</TabsTrigger>
        <TabsTrigger value="yarn"><Yarn className="size-4" /> yarn</TabsTrigger>
      </TabsList>
      <TabsContent value="npm">
        ```bash title="Terminal"
        cd my-api
        npm run dev
        ```
      </TabsContent>
      <TabsContent value="pnpm">
        ```bash title="Terminal"
        cd my-api
        pnpm dev
        ```
      </TabsContent>
      <TabsContent value="yarn">
        ```bash title="Terminal"
        cd my-api
        yarn dev
        ```
      </TabsContent>
    </Tabs>

    The dev server starts on `http://localhost:3000` with hot reload via `tsx watch`.
  </Step>
  <Step>
    ### Build and run

    <Tabs defaultValue="npm">
      <TabsList>
        <TabsTrigger value="npm"><NPM className="size-4" /> npm</TabsTrigger>
        <TabsTrigger value="pnpm"><Pnpm className="size-4" /> pnpm</TabsTrigger>
        <TabsTrigger value="yarn"><Yarn className="size-4" /> yarn</TabsTrigger>
      </TabsList>
      <TabsContent value="npm">
        ```bash title="Terminal"
        npm run build
        npm start
        ```
      </TabsContent>
      <TabsContent value="pnpm">
        ```bash title="Terminal"
        pnpm build
        pnpm start
        ```
      </TabsContent>
      <TabsContent value="yarn">
        ```bash title="Terminal"
        yarn build
        yarn start
        ```
      </TabsContent>
    </Tabs>
  </Step>
</Steps>

## Routes

Your project comes with these routes out of the box:

| Route | Description |
|-------|-------------|
| `GET /` | Welcome message with API version and environment |
| `GET /health` | Health check |
| `GET /api/examples` | List all examples (CRUD) |
| `POST /api/examples` | Create an example (validated with zod) |
| `GET /api/examples/:id` | Get a single example |
| `DELETE /api/examples/:id` | Delete an example |
