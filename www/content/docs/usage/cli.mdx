## Usage

```bash title="Terminal"
npx pest-js-app [options]
```

## Options

| Flag | Description |
|------|-------------|
| `--name <name>` | Project name (kebab-case) |
| `--database <orm>` | `prisma`, `drizzle`, `typeorm`, or `none` |
| `--db-provider <db>` | `postgresql`, `mysql`, or `sqlite` |
| `--docker` | Enable Docker support |
| `--no-docker` | Disable Docker support |
| `--package-manager <pm>` | `npm`, `pnpm`, or `yarn` |
| `-y, --yes` | Use defaults, skip prompts |
| `-v, --version` | Print version |
| `-h, --help` | Print help |

## Non-interactive mode

Use `--yes` to skip all prompts. Combine with other flags to customize:

```bash title="Terminal"
npx pest-js-app --yes --name my-api
```

Defaults when using `--yes`:

| Option | Default |
|--------|---------|
| name | `my-app` |
| description | `A PEST.js application` |
| author | from `git config user.name` |
| database | `none` |
| db-provider | `postgresql` |
| docker | `true` if database is set, `false` otherwise |
| package-manager | `npm` |
| git | `true` |
| install | `true` |
